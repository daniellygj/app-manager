### Documentação da API:
Depois de rodar o projeto, acessar a url http://localhost:8080/api/swagger-ui.html


### Como rodar o projeto:
* #### Docker
    * Execute o comando
      ```
        mvn install
      
        docker-compose up -d
      ```
* #### Local
    * Execute os comandos
      ```
        mvn install
      
        mvn spring-boot:run
      ```
